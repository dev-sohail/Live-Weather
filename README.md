# Live Weather

> A modern weather web application that provides real-time weather information and forecasts for locations around the world.

🌐 **Live Demo:** https://dev-sohail.github.io/Live-Weather/

---

## 📖 Overview

Live Weather is a responsive web application that allows users to search for any city and instantly view current weather conditions along with additional meteorological information.

The project was built to strengthen frontend development skills while working with real-world weather APIs, asynchronous JavaScript, and responsive user interfaces.

---

## ✨ Features

- 🌍 Search weather by city
- 📍 Current weather conditions
- 🌡 Temperature display
- 💧 Humidity information
- 🌬 Wind speed
- ☁ Weather condition & description
- 📱 Fully responsive design
- ⚡ Fast and lightweight interface
- 🔄 Real-time API data

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
- Fetch API

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/dev-sohail/Live-Weather.git
```

Navigate into the project

```bash
cd Live-Weather
```

Open

```text
index.html
```

or run it using **VS Code Live Server**.
